## План фиксации проблемы сборки в release

### Проблема:
electron-builder не может удалить `app.asar` в `release/win-unpacked/resources/` из-за блокировки файла другим процессом

### Решение по этапам:

**1. Диагностика блокировки:**
- Найти процесс, блокирующий файл с помощью утилит
- Проверить Windows Search, антивирус, синхронизацию

**2. Очистка системы:**
- Очистить кэш electron-builder
- Завершить все лишние Node.js процессы
- Добавить исключения в ПО безопасности

**3. Исправление конфигурации:**
- Вернуть `output: "release"` в package.json
- Добавить параметры для избежания блокировок

**4. Тестирование:**
- Выполнить полную сборку в release
- Проверить работоспособность дистрибутива

### Результат:
Стабильная сборка в стандартной директории release согласно проектным стандартам

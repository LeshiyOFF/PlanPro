================================================================================
1. О ПРОГРАММЕ
================================================================================

PlanPro — десктопное приложение для управления проектами с поддержкой:

   - Диаграммы Ганта с критическим путём
   - Сетевого графика (PERT)
   - Структуры декомпозиции работ (WBS)
   - Управления ресурсами и календарями
   - Отчётности по проекту


СИСТЕМНЫЕ ТРЕБОВАНИЯ:

   - Windows 10/11 или Linux (Astra Linux SE)
   - 4 ГБ RAM (рекомендуется 8 ГБ)
   - 500 МБ свободного места на диске


ПОДДЕРЖИВАЕМЫЕ ЯЗЫКИ:

   - Русский
   - English

2. НАЧАЛО РАБОТЫ

ПЕРВЫЙ ЗАПУСК
--------------------------------------------------------------------------------

При запуске PlanPro открывается главное меню с кнопками для работы с проектами.

СКРИНШОТ: Главное меню при запуске с кнопками "Создать проект", "Открыть 
проект", "Открыть последний проект" (если был запущен проект ранее) и 
областью для перетаскивания файлов


ЭЛЕМЕНТЫ ГЛАВНОГО МЕНЮ:

   1. Создать проект
      Создание нового пустого проекта

   2. Открыть проект
      Открытие существующего файла .pod

   3. Открыть последний проект
      Быстрый доступ к последнему открытому проекту
      Примечание: кнопка отображается только если проект был открыт ранее


ОБЛАСТЬ ДЛЯ ПЕРЕТАСКИВАНИЯ ФАЙЛОВ (только Windows):

   - Перетащите файл .pod в окно приложения для быстрого открытия
   - На Linux данная функция отключена из-за особенностей операционной системы


СОЗДАНИЕ ПЕРВОГО ПРОЕКТА
--------------------------------------------------------------------------------

Шаг 1: Нажмите кнопку "Создать проект" в главном меню

Шаг 2: Автоматически создаётся пустой проект с названием "Новый проект"

Шаг 3: Добавьте первую задачу — нажмите кнопку "+" в панели инструментов

Шаг 4: Введите название задачи в таблице

Шаг 5: Установите длительность задачи (например, 5 дней)

Шаг 6: Сохраните проект через меню "Файл → Сохранить"

СКРИНШОТ: Диаграмма Ганта с 3-5 простыми задачами, отображающимися как 
горизонтальные полосы на временной шкале


ЭЛЕМЕНТЫ ИНТЕРФЕЙСА:

   - Боковая панель (слева) — переключение между представлениями
   - Рабочая область (центр) — отображение текущего представления
   - Панель инструментов (сверху) — действия для текущего представления
   - Строка состояния (снизу) — информация о проекте

СОЗДАНИЕ НОВОГО ПРОЕКТА

Быстрое открытие (только Windows): Перетащите файл .pod в окно приложения

СОХРАНЕНИЕ ПРОЕКТА

Сохранить как:

   - Сохраняет проект под новым именем


================================================================================
4. ДИАГРАММА ГАНТА
================================================================================


НАЗНАЧЕНИЕ
--------------------------------------------------------------------------------

Главное представление для планирования и управления задачами проекта с 
визуализацией на временной шкале.

СКРИНШОТ: Диаграмма Ганта с таблицей задач слева и временной шкалой справа, 
задачи отображаются как горизонтальные полосы


ЭЛЕМЕНТЫ ИНТЕРФЕЙСА
--------------------------------------------------------------------------------

Таблица задач (левая часть):
   - ID задачи
   - Название
   - Дата начала
   - Дата окончания
   - Длительность
   - Прогресс (%)
   - Предшественники
   - Резерв

Временная шкала (правая часть):
   - Горизонтальные полосы задач
   - Линии связей между задачами
   - Вехи (ромбы)
   - Индикатор текущей даты (вертикальная линия)


ПАНЕЛЬ ИНСТРУМЕНТОВ
--------------------------------------------------------------------------------

СКРИНШОТ: Панель инструментов Ганта с кнопками: счётчик задач, переключатель 
масштаба (День/Неделя/Месяц), стрелки навигации, выбор даты, увеличить 
масштаб, уменьшить масштаб, сбросить масштаб, настройки, сохранить базовый 
план, критический путь

Элементы:

   1. Счётчик задач
      Отображает количество задач в проекте

   2. Масштаб времени
      Переключение между режимами: День / Неделя / Месяц

   3. Навигация
      Стрелки "Назад" / "Вперёд" для перемещения по времени

   4. Выбор даты
      Календарь для перехода к конкретной дате

   5. Увеличить масштаб
      Увеличение масштаба временной шкалы

   6. Уменьшить масштаб
      Уменьшение масштаба временной шкалы

   7. Сбросить масштаб
      Вписать все задачи в экран

   8. Настройки
      Настройки диаграммы

   9. Закрепить
      Сохранение базового плана для сравнения

   10. Критический путь
       Включение/выключение отображения критического пути


РАБОТА С ЗАДАЧАМИ
--------------------------------------------------------------------------------

ДОБАВЛЕНИЕ ЗАДАЧИ

   Действие: Нажмите кнопку "+"
   
   Результат: Добавляется новая задача в конец списка
   
   СКРИНШОТ: Новая задача в таблице с возможностью редактирования названия


РЕДАКТИРОВАНИЕ ЗАДАЧИ

   Название: 
      Кликните на ячейку с названием и введите текст
   
   Даты: 
      Кликните на ячейку с датой, откроется календарь
   
   Длительность: 
      Введите число (например, "5" для 5 дней)
   
   Прогресс: 
      Введите процент (от 0 до 100)


УДАЛЕНИЕ ЗАДАЧИ

   Действие: Выберите задачу → правый клик → "Удалить задачу"
   
   Важно: Удаление суммарной задачи удалит все её подзадачи


ПЕРЕМЕЩЕНИЕ ЗАДАЧИ НА ВРЕМЕННОЙ ШКАЛЕ

   Действие: Перетащите полосу задачи влево или вправо
   
   Результат: Изменяются даты начала и окончания задачи
   
   СКРИНШОТ: Курсор мыши захватывает полосу задачи для перетаскивания


ИЗМЕНЕНИЕ ДЛИТЕЛЬНОСТИ

   Действие: Перетащите правый край полосы задачи
   
   Результат: Изменяется дата окончания задачи


ИЗМЕНЕНИЕ ПРОГРЕССА ВЫПОЛНЕНИЯ

   Действие: Перетащите маркер внутри полосы задачи
   
   Результат: Изменяется процент выполнения (индикатор прогресса внутри 
   задачи меняет длину)
   
   СКРИНШОТ: Полоса задачи с индикатором прогресса внутри, частично заполненным


ИЕРАРХИЯ ЗАДАЧ
--------------------------------------------------------------------------------

СОЗДАНИЕ ПОДЗАДАЧИ

   Действие: Выберите задачу → Правый клик → "Увеличить отступ"
   
   Результат: Задача становится подзадачей предыдущей задачи
   
   Условие: Над выбранной задачей должна быть другая задача (будущий родитель)
   
   СКРИНШОТ: Иерархия задач с отступами (подзадачи смещены вправо относительно 
   родительской)


ПОВЫШЕНИЕ УРОВНЯ

   Действие: Выберите подзадачу → Правый клик → "Уменьшить отступ"
   
   Результат: Задача поднимается на уровень выше


СУММАРНАЯ ЗАДАЧА

   Создание:
      1. Создайте несколько задач
      2. Сделайте их подзадачами одной задачи (увеличение отступа)
      3. Родительская задача автоматически станет суммарной
   
   Особенности:
      - Суммарная задача отображается как серая полоса, охватывающая все 
        подзадачи
      - Длительность суммарной задачи рассчитывается автоматически
      - Даты начала и окончания определяются первой и последней подзадачей
   
   СКРИНШОТ: Суммарная задача (серая полоса) охватывает несколько подзадач


СВЯЗИ ЗАДАЧ (ЗАВИСИМОСТИ)
--------------------------------------------------------------------------------

СОЗДАНИЕ СВЯЗИ

   Шаг 1: Выберите задачу, которая зависит от другой
   
   Шаг 2: Правый клик → "Связать задачи"
   
   Шаг 3: Активируется режим связывания (доступные задачи подсвечиваются)
   
   Шаг 4: Кликните на задачу-предшественника
   
   Результат: Создаётся связь типа "Окончание-Начало"
   
   Отмена: Нажмите кнопку "Отмена"
   
   СКРИНШОТ: Режим связывания активен, доступные задачи выделены


ОТОБРАЖЕНИЕ СВЯЗЕЙ

   Визуализация: Стрелки между задачами на временной шкале
   
   Цвета:
      - Серый — обычная связь
      - Красный — связь на критическом пути
   
   СКРИНШОТ: Стрелка зависимости от конца одной задачи к началу другой


УДАЛЕНИЕ СВЯЗИ

   Действие: Выберите задачу → Правый клик → "Разорвать связи"
   
   Результат: Удаляются все предшественники выбранной задачи


ОГРАНИЧЕНИЯ СВЯЗЫВАНИЯ

   Запрещено:
      - Связать задачу с самой собой
      - Связать суммарную задачу с другой суммарной
      - Связать родительскую задачу с её дочерней (циклическая зависимость)
   
   Диалог: При попытке недопустимой связи появляется диалог с объяснением 
   причины
   
   СКРИНШОТ: Диалог "Связь заблокирована" с объяснением причины и названиями 
   задач


ВЕХА
--------------------------------------------------------------------------------

Назначение: Ключевая точка проекта с нулевой длительностью

Создание: Выберите задачу → Правый клик → "Сделать вехой"

Визуализация: Отображается как ромб вместо полосы

Возврат: Правый клик → "Сделать обычной задачей"

СКРИНШОТ: Веха отображается как ромб на временной шкале


КРИТИЧЕСКИЙ ПУТЬ
--------------------------------------------------------------------------------

Включение: Нажмите кнопку "Критический путь" (иконка пульса) на панели 
инструментов

Визуализация:
   - Задачи на критическом пути окрашиваются в красный цвет
   - Связи критического пути отображаются красными стрелками

Расчёт: Автоматически пересчитывается при изменении задач

Резерв: Отображается в колонке "Резерв" таблицы задач

СКРИНШОТ: Критический путь выделен красным цветом (задачи и связи)


БАЗОВЫЙ ПЛАН
--------------------------------------------------------------------------------

Создание: Нажмите кнопку "Закрепить" на панели инструментов

Назначение: Сохранение снимка текущего состояния проекта для сравнения

Использование: Базовый план используется в представлении "Гантт отслеживания"


КОНТЕКСТНОЕ МЕНЮ ЗАДАЧИ
--------------------------------------------------------------------------------

Вызов: Правый клик на задаче в таблице или на временной шкале

Пункты меню:

   1. Информация о задаче — открыть диалог свойств
   2. Сделать вехой / Сделать обычной задачей — переключение типа
   3. Связать задачи — активировать режим связывания
   4. Разорвать связи — удалить все предшественники
   5. Прервать задачу — разделение задачи (если включено в настройках)
   6. Увеличить отступ — сделать подзадачей
   7. Уменьшить отступ — поднять уровень
   8. Переместить вверх — изменить порядок в списке
   9. Переместить вниз — изменить порядок в списке
   10. Удалить задачу — удалить из проекта

СКРИНШОТ: Контекстное меню задачи с перечисленными пунктами


МАСШТАБИРОВАНИЕ ВРЕМЕНИ
--------------------------------------------------------------------------------

Режимы масштаба:
   - День — каждый день отображается отдельно
   - Неделя — группировка по неделям
   - Месяц — группировка по месяцам

Переключение: Кнопки в панели инструментов или кнопки увеличения/уменьшения 
масштаба

СКРИНШОТ: Переключатель масштаба "День/Неделя/Месяц"


НАВИГАЦИЯ ПО ВРЕМЕНИ
--------------------------------------------------------------------------------

Стрелки "Назад" / "Вперёд": Перемещение временной шкалы

Выбор даты: Клик на календарь → выбор конкретной даты

Ограничения: Навигация ограничена диапазоном проекта 
(минимальная/максимальная дата)

Индикатор "Сегодня": Вертикальная линия на временной шкале

СКРИНШОТ: Календарь выбора даты с ограничениями минимальной и максимальной 
даты


ДИАЛОГ СВОЙСТВ ЗАДАЧИ
--------------------------------------------------------------------------------

Открытие: Двойной клик на задаче или контекстное меню → "Информация о задаче"

Вкладки:
   1. Общие — название, даты, длительность, прогресс
   2. Предшественники — список предшествующих задач
   3. Ресурсы — назначенные ресурсы
   4. Заметки — текстовые заметки к задаче
   5. Дополнительно — дополнительные параметры

СКРИНШОТ: Диалог свойств задачи с 5 вкладками


================================================================================
5. СЕТЕВОЙ ГРАФИК (PERT)
================================================================================


НАЗНАЧЕНИЕ
--------------------------------------------------------------------------------

Визуализация зависимостей между задачами в виде узлов и связей.

СКРИНШОТ: Сетевой график с узлами-прямоугольниками, соединёнными стрелками


ЭЛЕМЕНТЫ УЗЛА
--------------------------------------------------------------------------------

Отображение:
   - Название задачи
   - Длительность (X дн.)
   - Даты начала и окончания
   - Индикатор прогресса выполнения

Типы узлов:
   - Обычная задача — прямоугольник с белым фоном
   - Суммарная задача — прямоугольник с возможностью сворачивания
   - Критическая задача — прямоугольник с красной заливкой

СКРИНШОТ: Узел задачи с названием "Задача 1", длительностью "5 дн.", датами 
и индикатором прогресса


СВЯЗИ МЕЖДУ УЗЛАМИ
--------------------------------------------------------------------------------

Визуализация: Стрелки между узлами

Цвета:
   - Серый — обычная связь
   - Красный — связь на критическом пути

СКРИНШОТ: Связи между узлами с направлением стрелок


ИЕРАРХИЯ И ГРУППИРОВКА
--------------------------------------------------------------------------------

Сворачивание группы:
   - Клик на суммарной задаче
   - Дочерние узлы скрываются
   - Иконка со стрелкой указывает на свёрнутое состояние

Разворачивание:
   - Повторный клик на суммарной задаче

СКРИНШОТ: Суммарная задача со свёрнутыми дочерними узлами и иконкой стрелки


ИНТЕРАКТИВНОСТЬ
--------------------------------------------------------------------------------

ПЕРЕТАСКИВАНИЕ УЗЛОВ

   Действие: Перетащите узел в любое место графа
   
   Результат: Узел перемещается, связи автоматически перестраиваются
   
   СКРИНШОТ: Курсор захватывает узел для перетаскивания


СОЗДАНИЕ СВЯЗИ

   Действие: Перетащите от одного узла к другому
   
   Результат: Создаётся связь типа "Окончание-Начало"


ОТКРЫТИЕ СВОЙСТВ ЗАДАЧИ

   Действие: Двойной клик на узле
   
   Результат: Открывается диалог свойств задачи (5 вкладок)


ВАЛИДАЦИЯ СВЯЗЕЙ
--------------------------------------------------------------------------------

Запрещено:
   - Связать узел с самим собой
   - Связать суммарные задачи
   - Создать циклическую зависимость (родитель → ребёнок)

Диалог: При блокировке связи появляется диалог с объяснением

Типы причин блокировки:
   - Попытка связать задачу с самой собой
   - Попытка связать суммарные задачи
   - Попытка создать цикл (родитель → ребёнок)

СКРИНШОТ: Диалог "Связь заблокирована" с объяснением причины


МАСШТАБИРОВАНИЕ
--------------------------------------------------------------------------------

Увеличить масштаб: Кнопка "+" — увеличить масштаб

Уменьшить масштаб: Кнопка "-" — уменьшить масштаб

Сбросить масштаб: Кнопка "Развернуть" — сбросить масштаб

Диапазон: От 0.5x до 2x

СКРИНШОТ: Панель инструментов с кнопками увеличить, уменьшить, сбросить 
масштаб


ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ
--------------------------------------------------------------------------------

Добавление задачи: Кнопка "+" — добавить новую задачу

Автоматическая раскладка: Кнопка "Автоматическая раскладка" — сбросить 
позиции узлов к начальной раскладке

Легенда: Нижняя панель с подсказками (красный = критический путь, обычная 
задача)

ФОРМАТЫ ФАЙЛОВ
--------------------------------------------------------------------------------

Собственный формат:
   - .pod (ProjectLibre Open Document)

Импорт:
   - Microsoft Project XML
   - Primavera XER
   - Microsoft Project MPP (через конвертацию)

Экспорт:
   - Microsoft Project XML
   - Primavera XER
   - PDF (отчёты)
   - Excel (таблицы)
   - CSV (данные)
   - PNG (диаграммы)


ИМПОРТ ИЗ MICROSOFT PROJECT
--------------------------------------------------------------------------------

Действие: Меню "Файл → Импорт → Microsoft Project"

Форматы: XML, MPP

Особенности конвертации:
   - Преобразование задач и зависимостей
   - Конвертация ресурсов и назначений
   - Сохранение календарей
   - Возможна потеря специфичных для MS Project данных


ИМПОРТ ИЗ PRIMAVERA
--------------------------------------------------------------------------------

Действие: Меню "Файл → Импорт → Primavera"

Формат: XER

Особенности:
   - Преобразование активностей в задачи
   - Конвертация ресурсов
   - Сохранение связей


ЭКСПОРТ В MICROSOFT PROJECT
--------------------------------------------------------------------------------

Действие: Меню "Файл → Экспорт → Microsoft Project XML"

Формат: XML (совместим с MS Project)


ЭКСПОРТ В PRIMAVERA
--------------------------------------------------------------------------------

Действие: Меню "Файл → Экспорт → Primavera XER"

Формат: XER


ЭКСПОРТ В PDF
--------------------------------------------------------------------------------

Доступно в:
   - Отчёты

Действие: Кнопка "Экспорт PDF"


ЭКСПОРТ В EXCEL/CSV
--------------------------------------------------------------------------------

Доступно в:
   - Лист задач
   - Лист ресурсов
   - Использование задач
   - Использование ресурсов

Действие: Кнопка "Экспорт CSV"

Формат: CSV (открывается в Excel)


ЭКСПОРТ ДИАГРАММ В PNG
--------------------------------------------------------------------------------

Доступно в:
   - Структура декомпозиции работ (WBS)

Действие: Кнопка "Экспорт" (иконка загрузки)

Формат: PNG (высокое качество, масштаб 2x)

Имя файла: Автоматическое (WBS_YYYY-MM-DD.png)
